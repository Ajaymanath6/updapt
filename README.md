# Updapt

Enterprise ESG (Environmental, Social, and Governance) Management Platform

## Features

- **Site-Centric Assignment Matrix**: Powerful bulk assignment system with checkboxes
- **At-a-Glance User Visibility**: See assigned users with colored avatars
- **Bulk Operations**: Assign or remove users from multiple combinations at once
- **Smart Filtering**: Multi-select dropdowns for sites and metrics
- **Professional UI**: Modern, clean interface with world-class UX

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Remix Icons

## Getting Started

### Prerequisites

- Node.js 16+ and npm

### Installation

```bash
# Install dependencies
cd cbp-client
npm install
```

### Development

```bash
# Start development server
npm run dev
```

The application will be available at `http://localhost:5173`

### Build

```bash
# Build for production
npm run build
```

### Preview Production Build

```bash
# Preview production build locally
npm run preview
```

## Deployment

This project is configured for automatic deployment to GitHub Pages via GitHub Actions.
